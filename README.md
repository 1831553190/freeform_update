稳定运行意味着你打开QQ或者WhatApp这类应用时，应用能稳定的在小窗中打开


**这个项目的应用有什么特殊的问题（bug）存在吗**

1. 旋转屏幕（自行体会）
2. 大小缩放，仅支持同比例缩放，不支持改宽高
3. 其他bug应该和官方发行的版本bug一致
